# TonyPi-LLM-Powered-Voice

## 仓库介绍

本仓库旨在对 TonyPi 人形机器人进行功能升级，通过接入大模型技术，实现语音交互控制。我们致力于优化现有功能，打造更智能、更便捷的人机交互体验。
